# 월간 일본어 복습 테스트 생성기

한 달간 만든 [핸드아웃](HANDOUT.md) 내용을 바탕으로 단어 시험지를 만들고 PDF로 저장하는 도구입니다. 빌드 과정 없는 순수 HTML/CSS/JS이며, `Thumbnail-Generator` 저장소의 `test.html`에서 바로 실행합니다.

## 실행 방법

```bash
# macOS
open test.html

# Windows
start test.html
```

핸드아웃 생성기(`handout.html`)·썸네일 생성기(`index.html`) 상단의 **복습 테스트 만들기 →** 버튼으로도 접근할 수 있습니다.

> **참고:** html2pdf.js와 Google Fonts는 CDN에서 로드되므로 최초 실행 시 인터넷 연결이 필요합니다.

## 화면 구성

좌측 폼에 입력하면 우측 A4 미리보기에 실시간으로 반영됩니다.

| 항목 | 설명 |
|------|------|
| 복습할 월 | 입력한 값이 미리보기 제목에 `{월} 일본어 복습하기` 형식으로 표시됨 (예: "7월" → "7월 일본어 복습하기") |
| ① 단어 시험 | 문제(뜻 또는 단어)를 여러 개 추가. 2열로 배치되어 한 페이지에 많이 들어감 |
| ② 예문 빈칸 채우기 | 빈칸이 포함된 일본어 문장 + 선택 힌트를 여러 개 추가 |
| ③ 예문 해석하기 | 일본어 문장을 여러 개 추가하면 각 문장 아래 답을 쓸 빈 줄이 표시됨 |
| ④ 복습 문장 작문하기 | 작문 주제/조건을 여러 개 추가하면 각 문항 아래 두 줄의 작문 공간이 표시됨 |

### 문제 추가하기

각 영역 하단의 **+ 문제 추가** 버튼을 누르면 입력 카드가 하나씩 추가됩니다. 카드가 2개 이상일 때는 각 카드 우측 상단의 **✕** 버튼으로 개별 삭제할 수 있습니다.

### 빈 항목 자동 숨김

내용을 입력하지 않은 영역(②③④)은 제목·구분선을 포함해 미리보기와 PDF에서 자동으로 숨겨집니다.

### 한 페이지 분량 안내

핵심 단어 위주로 구성하면(단어 시험 8~10개 + 나머지 유형 2~4개씩) A4 한 장에 넉넉히 들어갑니다. 네 가지 유형을 모두 최대치(단어 20개 이상 + 나머지 각 6~8개)로 채우면 한 페이지를 넘어가 내용이 잘릴 수 있으니, 한 달 분량이 많다면 문제 유형별로 나누어 여러 장으로 출력하는 것을 권장합니다.

## PDF 저장

**PDF 다운로드** 버튼을 누르면 미리보기 내용을 A4 크기 그대로 PDF로 저장합니다.

- 파일명은 `Study_Japanese_Test_{복습할 월}.pdf` 형식으로 자동 생성됩니다.
- 저장 중에는 로딩 오버레이가 표시되며, 완료 후 자동으로 사라집니다.

## 관련 파일

| 파일 | 역할 |
|------|------|
| `test.html` | 복습 테스트 생성기 마크업 (폼 + 미리보기) |
| `css/test.css` | 테스트지 전용 스타일 (`css/handout.css`의 A4 페이지·폼 스타일을 공유) |
| `js/test.js` | 폼 상태 관리, 미리보기 렌더링, PDF 내보내기 로직 |
