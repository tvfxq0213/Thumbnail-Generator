# NHK Easy News Handout Generator

**NHK Easy News**로 일본어를 공부할 때 쓰는 A4 학습 핸드아웃을 만들고 PDF로 저장하는 도구입니다. 빌드 과정 없는 순수 HTML/CSS/JS이며, `Thumbnail-Generator` 저장소의 `handout.html`에서 바로 실행합니다.

## 실행 방법

```bash
# macOS
open handout.html

# Windows
start handout.html
```

썸네일 생성기(`index.html`) 상단의 **핸드아웃 만들기 →** 버튼으로도 접근할 수 있습니다.

> **참고:** html2pdf.js와 Google Fonts는 CDN에서 로드되므로 최초 실행 시 인터넷 연결이 필요합니다.

## 화면 구성

좌측 폼에 입력하면 우측 A4 미리보기에 실시간으로 반영됩니다.

| 항목 | 설명 |
|------|------|
| 상단 브랜드명 / 날짜·회차 | 핸드아웃 헤더에 표시되는 텍스트 |
| ① 오늘의 핵심 단어 | 단어 / 뜻 / 예문을 여러 개 추가·삭제 가능 |
| ② 문법 | 문법 패턴 / 예문을 여러 개 추가·삭제 가능 |
| ③ 오늘의 복습 (선택 입력) | 복습 문장 최대 3개. 비워두면 해당 항목은 미리보기에서 자동으로 숨겨짐 |
| ④ 일본 문화 알아보기 | 일본 문화·풍습 관련 내용을 자유롭게 기록 |

### ① 핵심 단어 / ② 문법 — 여러 개 추가하기

각 영역 하단의 **+ 단어 추가** / **+ 문법 추가** 버튼을 누르면 입력 카드가 하나씩 추가됩니다. 카드가 2개 이상일 때는 각 카드 우측 상단의 **✕** 버튼으로 개별 삭제할 수 있습니다. 추가된 항목은 미리보기에 ①②③… 순서로 자동 번호가 매겨집니다.

### ③ 오늘의 복습 — 빈 항목 자동 숨김

복습 1~3 중 입력하지 않은 항목은 미리보기와 PDF에 표시되지 않습니다. 세 항목을 모두 비워두면 "오늘의 복습" 섹션 전체(제목·구분선 포함)가 나타나지 않습니다.

### 빈칸 연습 박스

단어·문법 예문을 입력하면 글자 수(공백 제외)에 맞춰 원고지 스타일의 빈칸 연습 박스가 자동으로 생성됩니다(최소 6칸 ~ 최대 20칸).

## PDF 저장

**PDF 다운로드** 버튼을 누르면 미리보기 내용을 A4 크기 그대로 PDF로 저장합니다.

- 파일명은 `Study_Japanese_{날짜 또는 첫 단어}.pdf` 형식으로 자동 생성됩니다.
- 저장 중에는 로딩 오버레이가 표시되며, 완료 후 자동으로 사라집니다.

## 관련 파일

| 파일 | 역할 |
|------|------|
| `handout.html` | 핸드아웃 생성기 마크업 (폼 + 미리보기) |
| `css/handout.css` | A4 미리보기·반복 입력 카드 스타일 |
| `js/handout.js` | 폼 상태 관리, 미리보기 렌더링, PDF 내보내기 로직 |
