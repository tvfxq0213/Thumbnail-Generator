# NHK Easy News Handout Generator

**NHK Easy News**로 일본어를 공부할 때 쓰는 A4 학습 핸드아웃을 만들고 PDF로 저장하는 도구입니다. 빌드 과정 없는 순수 HTML/CSS/JS이며, `Thumbnail-Generator` 저장소의 `handout.html`에서 바로 실행합니다.

## 실행 방법

```bash
# macOS
open handout.html

# Windows
start handout.html
```

썸네일 생성기(`index.html`) 상단의 **핸드아웃 만들기 →** 버튼으로도 접근할 수 있습니다.

> **참고:** html2pdf.js와 Google Fonts는 CDN에서 로드되므로 최초 실행 시 인터넷 연결이 필요합니다.

## 화면 구성

좌측 폼에 입력하면 우측 A4 미리보기에 실시간으로 반영됩니다.

| 항목 | 설명 |
|------|------|
| 상단 브랜드명 / 회차 번호 | 브랜드명과 `No.001` 형식으로 고정 표시되는 회차 번호 |
| 오늘의 난이도 | 별점(1~5) 클릭으로 선택. 같은 별을 다시 누르면 초기화되고, 설정하지 않으면 미리보기에서 숨겨짐 |
| ① 오늘의 핵심 단어 | 단어 / 뜻 / 예문 / 예문 해석을 여러 개 추가·삭제 가능 (최대 10개 안팎까지 한 페이지에 들어가도록 설계됨) |
| ② 문법 | 문법 패턴 / 예문 / 예문 해석을 여러 개 추가·삭제 가능 (2~3개 기준으로 설계됨) |
| ③ 오늘의 핵심 문장 (선택 입력) | 문장 최대 3개. 비워두면 해당 항목은 미리보기에서 자동으로 숨겨짐 |
| ④ 일본 문화 알아보기 | 일본 문화·풍습 관련 내용을 자유롭게 기록. 비워두면 섹션 전체가 숨겨짐 |
| ⑤ QR 코드 (선택) | 블로그 게시물로 연결되는 QR 코드 이미지를 업로드하면 페이지 우측 하단에 표시 |

### ① 핵심 단어 / ② 문법 — 여러 개 추가하기

각 영역 하단의 **+ 단어 추가** / **+ 문법 추가** 버튼을 누르면 입력 카드가 하나씩 추가됩니다. 카드가 2개 이상일 때는 각 카드 우측 상단의 **✕** 버튼으로 개별 삭제할 수 있습니다. 추가된 항목은 미리보기에 ①②③… 순서로 자동 번호가 매겨집니다. 단어 영역은 한 페이지에 여러 개가 들어가도록 2열로 표시됩니다.

### ③ 오늘의 핵심 문장 / ④ 일본 문화 알아보기 — 빈 항목 자동 숨김

내용을 입력하지 않으면 해당 섹션(제목·구분선 포함)이 미리보기와 PDF에서 자동으로 숨겨집니다.

### 예문 해석

단어·문법의 예문 옆에 한국어 해석이 함께 표시됩니다.

### 암기 확인 체크 박스

단어 옆에 체크 박스 3칸이 표시됩니다. 같은 단어를 여러 번 복습하며 외웠는지 손으로 체크하는 용도입니다.

### 일본 문화 알아보기 — 굵게·줄바꿈 태그

입력창에 `<b>텍스트</b>`(굵게) 또는 `<br>`(줄바꿈) 태그를 넣으면 미리보기·PDF에 그대로 서식이 적용됩니다. 이 두 태그(및 `<strong>`) 외의 다른 태그는 모두 일반 텍스트로만 표시되어 안전합니다.

### QR 코드

업로드한 이미지는 저장 없이 브라우저 안에서만 미리보기에 반영되며, 페이지 우측 하단에 고정 크기로 표시됩니다.

## PDF 저장

**PDF 다운로드** 버튼을 누르면 미리보기 내용을 A4 크기 그대로 PDF로 저장합니다.

- 파일명은 `Study_Japanese_{회차 번호 또는 첫 단어}.pdf` 형식으로 자동 생성됩니다.
- 저장 중에는 로딩 오버레이가 표시되며, 완료 후 자동으로 사라집니다.

## 관련 파일

| 파일 | 역할 |
|------|------|
| `handout.html` | 핸드아웃 생성기 마크업 (폼 + 미리보기) |
| `css/handout.css` | A4 미리보기·반복 입력 카드 스타일 |
| `js/handout.js` | 폼 상태 관리, 미리보기 렌더링, PDF 내보내기 로직 |
