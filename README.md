# Thumbnail Generator

네이버 블로그용 1200×1200 썸네일을 빠르게 제작하는 웹 기반 도구입니다. 상단 탭으로 세 가지 썸네일 유형을 전환하며 사용합니다.

- **일본어 학습** — **NHK Easy News** 시리즈용 무인양품(MUJI) 교재 느낌의 심플한 텍스트 썸네일
- **여행 콘텐츠** — 업로드한 사진 위에 딤(dim) 처리 후 텍스트를 올리는 여행기/정보형(Q&A) 썸네일
- **책 콘텐츠** — "○○권 추천" 류의 책 리스트/에세이 포스팅에 어울리는, 권수 배지 + 제목 + 블로그 주소가 들어간 텍스트 썸네일

같은 저장소에서 **NHK Easy News 학습용 A4 핸드아웃(PDF) 생성기**(`handout.html`)와 **월간 일본어 복습 테스트 생성기**(`test.html`)도 함께 제공합니다. 사용법은 각각 [HANDOUT.md](HANDOUT.md), [TEST.md](TEST.md) 참고.

빌드 과정 없는 순수 HTML/CSS/JS라 로컬 실행은 물론 **GitHub Pages로도 바로 배포**할 수 있습니다.

## 설치 방법

별도의 빌드 과정이나 패키지 설치가 필요 없습니다.

1. 이 저장소를 클론하거나 ZIP으로 다운로드합니다.

```bash
git clone https://github.com/tvfxq0213/Thumbnail-Generator.git
cd Thumbnail-Generator
```

2. `index.html`을 브라우저에서 엽니다.

```bash
# macOS
open index.html

# Windows
start index.html
```

> **참고:** html2canvas와 Google Fonts는 CDN에서 로드됩니다. 최초 실행 시 인터넷 연결이 필요합니다.
> 모든 내부 리소스 경로(`css/style.css`, `js/app.js`)는 상대경로로 작성되어 있어, 로컬 파일 실행·서브폴더 배포(GitHub Pages 등) 어디서든 동일하게 동작합니다.

## GitHub Pages로 배포하기

1. GitHub 저장소 → **Settings → Pages** 이동
2. **Source**를 `Deploy from a branch`로 설정
3. **Branch**를 `main` / `(root)`로 선택 후 저장
4. 잠시 후 아래 주소에서 확인 가능합니다.

```
https://tvfxq0213.github.io/Thumbnail-Generator/
```

> 별도의 빌드 스텝이 없으므로 `main` 브랜치에 푸시만 하면 몇 분 내로 반영됩니다.

## 사용 방법

### 공통

- 상단 탭에서 **일본어 학습 / 여행 콘텐츠 / 책 콘텐츠** 유형을 선택합니다.
- **🌙** 버튼으로 다크모드를, **컬러 테마** 버튼으로 포인트 색상을 변경할 수 있습니다.
- **PNG 저장** 또는 **JPG 저장** 버튼은 현재 선택된 탭의 썸네일을 저장합니다.

### 일본어 학습 탭

1. **한국어 제목**과 **일본어 제목**을 입력합니다.
2. **카테고리**(JLPT N5~N1, 여행 일본어, 정치/경제/사회/문화 등)와 **회차번호**를 입력합니다.
   - 회차 `23` → 미리보기에 `#023`으로 표시됩니다.
3. 우측 미리보기에서 실시간으로 결과를 확인 후 저장합니다.
   - 파일명 예: `023_정시에출발하다.png`

### 여행 콘텐츠 탭

1. **배경 이미지**를 업로드합니다.
2. **템플릿 스타일**을 선택합니다.
   - **여행기(브랜드형)**: 국가/브랜드 라벨 + 큰 제목 + 강조색 부제 + 시리즈 번호(Day 04 등)
   - **정보형(Q&A)**: 국가/브랜드 라벨 + 질문형 제목 + 담백한 부제(후기 등)
3. **딤(dim) 처리 색상**(검정/흰색)과 **딤 농도**를 조절해 이미지 위 텍스트 가독성을 맞춥니다.
4. **국가/브랜드 라벨**, **제목**, **부제**, **시리즈 번호**(선택), **블로그 주소**를 입력합니다.
5. **블로그 주소 위치**(좌상단/우상단/좌하단/우하단)를 선택합니다.
6. 저장 버튼으로 PNG/JPG를 내려받습니다.

### 책 콘텐츠 탭

1. **카테고리 라벨**(선택, 예: "여름 소설 추천")과 **제목**을 입력합니다.
2. **권수**(선택)와 **단위**(기본값 "권", "선"/"편" 등으로 변경 가능)를 입력하면 제목 위에 강조 배지로 표시됩니다.
3. **블로그 주소**와 **위치**(좌상단/우상단/좌하단/우하단)를 입력합니다.
4. 저장 버튼으로 PNG/JPG를 내려받습니다.

## 기능 설명

| 기능 | 설명 |
|------|------|
| 모드 탭 전환 | 일본어 학습 / 여행 콘텐츠 / 책 콘텐츠 썸네일을 한 페이지에서 전환 |
| 실시간 미리보기 | 입력 즉시 썸네일에 반영 |
| 자동 폰트 크기 | 제목 길이에 따라 자동 축소·확대 (한국어 48~78px, 일본어 42~56px, 여행 제목 66~130px) |
| 자동 줄바꿈 | 긴 제목은 2~3줄로 자동 배치 (`line-break`, `word-break` 활용) |
| 회차 포맷 | 001, 023, 125 등 3자리 `#` 형식 자동 변환 |
| 배경 이미지 업로드 | 여행 탭에서 사진을 업로드해 배경으로 사용 (FileReader 기반, 서버 업로드 없음) |
| 딤(dim) 처리 | 검정/흰색 오버레이 + 농도(0~90%) 슬라이더로 가독성 조절 |
| 템플릿 스타일 | 여행기(브랜드형) / 정보형(Q&A) 레이아웃 토글 |
| 블로그 주소 워터마크 | 4모서리 중 위치 선택 가능, 비워두면 자동 숨김 |
| 컬러 테마 | Green, Blue, Orange, Red, Purple, Gray |
| 다크모드 | Light / Dark 원클릭 전환 |
| 고해상도 저장 | html2canvas 기반 1200×1200 원본 해상도 저장 |
| PNG 로딩 표시 | PNG 저장 전 1초 로딩 애니메이션 |
| localStorage | 카테고리·테마·모드 등 마지막 설정 자동 저장·복원 |
| 반응형 | 데스크톱(좌 입력 / 우 미리보기), 모바일(세로 배치) |

## 사용 라이브러리

| [html2canvas](https://html2canvas.hertzen.com/) v1.4.1 | 썸네일 이미지 변환 | jsDelivr CDN |
| [html2pdf.js](https://github.com/eKoopmans/html2pdf.js) v0.10.1 | 핸드아웃 A4 PDF 변환 | jsDelivr CDN |
| [Pretendard](https://github.com/orioncactus/pretendard) | 한국어 폰트 | jsDelivr CDN |
| [Noto Sans JP](https://fonts.google.com/noto/specimen/Noto+Sans+JP) | 일본어 폰트 | Google Fonts |
